# Novatrum OS: Digital Infrastructure and Agency Management System

**Novatrum OS** is the primary operational backbone of **Novatrum Infrastructure and Engineering**. It is a proprietary, integrated system designed to manage high-end web development projects, IT solution architectures, and client financial lifecycles. This platform represents the technical standard and operational excellence of Novatrum.

## System Overview
This system serves as a **centralized command center**, automating the transition from initial project discovery to final delivery. It ensures data integrity and operational transparency across all IT workflows and client interactions.

## Core Operational Modules

### 1. Project Discovery Engine
A technical blueprinting system that analyzes client requirements and provides immediate, **data-driven cost estimates** for custom web and IT solutions.

### 2. Financial Ledger and Document Automation
An advanced invoicing system featuring **server-side PDF generation**. It handles project installments, maintenance subscriptions, and real-time profitability tracking with high precision.

### 3. Client Workspace
A secure, encrypted portal where clients can monitor project milestones, access their financial history, and manage official documentation.

### 4. Operational Intelligence
Deep integration with financial nodes to automate **net taxable income calculations**, expense tracking, and profit margin analysis.

## Technical Architecture
The platform utilizes a high-performance stack to ensure maximum security and scalability:

* **Frontend:** Next.js 14 (App Router)
* **Backend:** Node.js API Routes (Server-Side Document Processing)
* **Database:** Supabase PostgreSQL with strict Row Level Security (RLS)
* **Communication:** Resend API for automated enterprise messaging

# Novatrum OS: Dijital Altyapı ve Ajans Yönetim Sistemi

**Novatrum OS**, **Novatrum Altyapı ve Mühendislik** şirketinin temel operasyonel omurgasıdır. Üst düzey web geliştirme projelerini, IT çözüm mimarilerini ve müşteri finansal döngülerini yönetmek için tasarlanmış, şirkete özel entegre bir sistemdir. Bu platform, Novatrum'un teknik standartlarını ve operasyonel mükemmelliğini temsil eder.

## Sisteme Genel Bakış
Bu sistem, merkezi bir **komuta merkezi** görevi görerek ilk proje keşif aşamasından son teslimata kadar olan süreci otomatize eder. Tüm IT iş akışlarında ve müşteri etkileşimlerinde veri bütünlüğü ve operasyonel şeffaflık sağlar.

## Temel Operasyonel Modüller

### 1. Proje Keşif Motoru
Müşteri gereksinimlerini analiz eden ve özel web ve IT çözümleri için veriye dayalı anlık **teknik maliyet tahminleri** sunan yapılandırma sistemidir.

### 2. Finansal Defter ve Belge Otomasyonu
**Sunucu taraflı PDF üretimi** yapan gelişmiş bir faturalandırma sistemidir. Proje taksitlerini, bakım aboneliklerini ve gerçek zamanlı kârlılık takibini yüksek hassasiyetle yönetir.

### 3. Müşteri Çalışma Alanı
Müşterilerin proje kilometre taşlarını izleyebileceği, finansal geçmişlerine erişebileceği ve resmi belgeleri yönetebileceği güvenli ve şifreli bir portaldır.

### 4. Operasyonel Zeka
Net **vergi matrahı hesaplamalarını**, gider takibini ve kâr marjı analizlerini otomatize etmek için finansal düğümlerle derin entegrasyon sağlar.

## Teknik Mimari
Platform, maksimum güvenlik ve ölçeklenebilirlik sağlamak için yüksek performanslı bir teknoloji yığını kullanır:

* **Frontend:** Next.js 14 (App Router)
* **Backend:** Node.js API Routes (Sunucu Taraflı Belge İşleme)
* **Veritabanı:** Sıkı Satır Düzeyinde Güvenlik (RLS) politikalarına sahip Supabase PostgreSQL
* **İletişim:** Kurumsal mesajlaşma otomasyonu için Resend API
